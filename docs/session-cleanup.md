# 会话清理配置

## 功能说明
cc-pet-web项目包含了一个自动会话清理功能，每天自动清理10天没有交互的会话。

## 配置参数
- **自动清理周期**: 每24小时运行一次
- **会话保留期限**: 10天（可根据需要调整）
- **清理条件**: 最后活跃时间超过指定天数的会话将被删除

## 运行方式
1. **自动运行**: 服务启动后自动开始定期清理任务
2. **手动运行**: 可通过以下命令手动执行清理：
   ```bash
   cd packages/server
   pnpm cleanup-sessions
   ```

## 实现原理
- 使用SQLite数据库查询`last_active_at`时间戳
- 比较当前时间与`last_active_at`时间差
- 删除超过设定天数阈值的会话记录
- 自动清理关联的消息记录（通过外键关系）

## 日志记录
每次清理操作会在控制台输出清理的会话数量，便于监控清理效果。

## 自定义配置
如需修改清理的天数阈值，可以通过修改源码中的数值来实现。
